# umux
Nano multiplexer

## Usage
```sh
umux - nano multiplexer

Options:
  -h, --help        Output usage information
  -v, --version     Output version number

Commands:
  (default)         create a local multiplexer
```

## Installation
```sh
$ cargo install umux
```
